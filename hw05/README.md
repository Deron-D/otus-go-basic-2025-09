# **Лекция №9: Выбор темы и организация проектной работы**
> _Golang_Developer_Basic_

## **Задание:**
Домашнее задание
Создаем структуры и полями нужных типов

Цель:
Научиться работать со структурами и полями;
Разобраться, как работать со строгой типизацией;
Понять, как объявлять поля, доступные только внутри пакета или снаружи.


Описание/Пошаговая инструкция выполнения домашнего задания:
Создать отдельные пакеты в internal/model, которые будут содержать все структуры, необходимые для реализации бизнес-логики вашего приложения;
С помощью структур и их полей необходимо описать все сущности, которые будут использоваться в вашем проекте, и сгруппировать их по пакетам, чтобы сильно связанные структуры в одном пакете, например для сущности book, и слабо связанные в другом типа order или account, каждую в своем;
Попробовать реализовать в структуре приватные переменные, которые можно отобразить снаружи пакета только с помощью метода структуры и которые можно изменить тоже только через метод.


Критерии оценки:
Реализована требуемая логика;

Статус ""Принято"" ставится при выполнении всех критериев


Компетенции:
Синтаксис языка Go
- работа со сложными типами
- научиться создавать собственные типы
- научиться создавать структуры с полями
- разобраться со встраиванием структур
- написание методов для структур
